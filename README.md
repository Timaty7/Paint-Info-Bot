# Paint-Info-Bot
AI-ассистент для интернет-магазина «Центр Красок #1» на базе Aiogram 3 и Google Gemini API. 
Автоматизирует консультацию клиентов: обрабатывает запросы по ассортименту, адресам, акциям и услугам колеровки строго на основе базы знаний компании. 
Реализована валидация ответов без Markdown и стабильная обработка ошибок

(Retry механизм при перегрузке API)
